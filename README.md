## 🚀 Getting Started

### Prerequisites

1. Clone the repo
   ```sh
   $ git clone https://github.com/yuvrajverma01/Dev-Circle-React-App.git
   ```
2. Install NPM packages
   ```sh
   $ npx create-react-app my-app
   ```
3. Change the directory
   ```sh
   $ cd my-app
   ```
